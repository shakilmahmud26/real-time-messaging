Welcome to Real-time messaging service

1.	Downlaod and install xampp server.
2.	Extract the file in the htdocs folder.
3.	Run xampp server, open phpmyadmin and create a database named 'real-msg'.
4.	Import the 'real-msg.sql' file in the database.
5.	Write this 'localhost/real-time-message' in the url bar.
6.	That's it :)
